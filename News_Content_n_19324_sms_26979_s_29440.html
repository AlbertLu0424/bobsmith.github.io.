2019年08月5~6日 「人口數據科學」工作坊：QGIS的人口地圖繪製
  • 發布單位：社會科學院

適合對象：不具有GIS的使用經驗，從基礎開始學習繪製人口地圖與地理資料視覺化分析

時間：2019年8/5 (週一)、8/6 (週二) 10:00am - 4:00pm 
地點：台大地理環境資源學系302電腦教室 (一人一機實作環境)

授課講師：溫在弘 教授 (台大地理環境資源學系) 
(台大地理環境資源學系主任、中華民國地圖學會秘書長)

 

活動內容

名額：40名 
費用：$3000 / person 
教材：課程講義、實習資料等 (QGIS為開源軟體，可免費下載與使用) 
課程內容：提供學員實作各類型人口地圖的展現方式 
上課時間：2019年8月5日（週一）上午10時至下午4時與 
2019年8月6日（週二）上午10時至下午4時 
（共2日。不提供午餐，可在附近福利社或餐廳用餐）） 
上課地點：國立台灣大學地理環境資源學系302電腦教室（台北市羅斯福路四段1號） 
交通： 
捷運：捷運松山新店線–公館站 2號出口 
開車：地理系館位置近基隆路，開車建議由基隆路四段144巷進入 

報名資訊：

  1. 報名期間尚未開始。名額40位，額滿為止。
  2. 請有興趣參加者至臺大活動報名系統填寫個人資料，至臺灣大學線上繳費系統列印繳款單後，至華南銀行或ATM繳費。
  3. 主辦單位收到報名費後，會與報名者聯繫，確認報名成功。

報名網址：https://my.ntu.edu.tw/actregister/sessionList.aspx?actID=20194100_01


聯絡人：國立臺灣大學人口與性別研究中心彭小姐 
連絡電話：(02) 3366-3400　E-Mail: psc@ntu.edu.tw

課程內容

講授繪製人口地圖的實作技術與方法 
8/5：1. QGIS概述與地圖基本原理 2. 基本人口地圖實作 
8/6：進階人口統計地圖 (圓餅圖、長條圖、流動與面積變形等)
基本人口地圖
點子圖 Dot Map 面量圖 Shaded Map (Choropleth Map) 
比例符號圖 Proportional Circle Map
進階的人口統計地圖
統計地圖 (圓餅圖) Pie Chart Map
統計地圖 (長條圖) Bar Chart Map
流動地圖 Flow Map
面積變形圖 Cartogram
